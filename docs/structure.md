# Project Structure

```
/ (repo root)
├── NexaroDesktopDMI/            # Nexacro N v24 project source
│   ├── common/
│   ├── design/
│   ├── frame/
│   ├── sample/
│   ├── samplefile/
│   └── _extlib_/
├── NexaroDesktopDMI_output/     # Generated output from Nexacro N v24 Studio

```

## Execution Paths (reference)

- `NexaroDesktopDMI_output/index.html`
  - Example: `http://localhost:8080/NexaroDesktopDMI_output/index.html`
- `NexaroDesktopDMI_output/quickview.html`
  - Example:
    `http://localhost:8080/NexaroDesktopDMI_output/quickview.html?screenid=desktop&formname=frame::frmLeft.xfdl`
